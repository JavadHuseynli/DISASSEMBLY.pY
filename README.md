# DISASSEMBLY.PY

Interactive disassembly analyzer for executable files.

## Features

- Disassemble executable files
- Interactive terminal UI
- Cross-reference analysis
- Instruction-level navigation
- Zoom and search capabilities

## Installation

```bash
cd exe_analyzer
pip install -r requirements.txt
```

## Usage

```bash
python main.py <executable_file>
```
